# DonkRFID-Clonner

## Necessary modules and stuff
1. Arduino Nano
2. RC522 RFID Module
3. 2 Buttons
4. 2x LED Diodes (RED and GREEN)
5. 2x Resistor with 220 Ohm
6. Bread Board
7. Some Wires (about 20pcs)

# Connection

## RFID Module
- SDA to D10
- SCK to D13
- MOSI to D11
- MISO to D12
- GND to GND
- VCC to 3.3V

## Buttons
- Buttton Write to D2
- Buttton Read to D3

## LED Diodes
- LED-GREEN to D4 ! with 220 Ohm RESISTOR !
- LED-RED to D5 ! with 220 Ohm RESISTOR !
